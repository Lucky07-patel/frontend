# Lead Capture Frontend

This is the frontend for the Lead Capture project built using React.js, Vite, and Tailwind CSS.

## Features

- Responsive lead capture form
- Fields:
  - Full Name
  - Email
  - Business Name
  - Message
- Client-side validation
- Error messages for invalid inputs
- Admin page to view submitted leads
- AI score color indicator:
  - Hot
  - Warm
  - Cold
  - Pending

## Tech Stack

- React.js
- Vite
- Tailwind CSS
